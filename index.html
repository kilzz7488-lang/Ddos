👤 Siapa itu Hacker?

Hacker adalah orang yang mempelajari sistem secara mendalam. Bukan cuma soal meretas, tapi juga memahami cara kerja teknologi.

💡 Di Indonesia, hacker etis justru dibutuhkan perusahaan buat ngetes keamanan.

⚫ Jenis Hacker

• White Hat: Hacker baik (legal)
• Black Hat: Hacker jahat (kriminal)
• Grey Hat: Di antaranya
• Script Kiddie: Newbie pake tools siap pakai

🛡️ Cyber Security

Keamanan siber itu menjaga data, sistem, dan jaringan dari serangan. Termasuk antisipasi virus, hacking, dan pencurian data.

🔒 "Keamanan bukan produk, tapi proses"

⚠️ Ancaman Sering Terjadi

• Phishing: Link palsu buat curi data
• Malware: Virus, trojan, ransomware
• DDoS: Serangan banjir traffic
• SQL Injection: Bobol database
• XSS: Suntik kode jahat ke website

🔧 Cara Lindungi Diri

1. Pake password kuat & beda tiap akun
2. Aktifkan 2FA (verifikasi 2 langkah)
3. Hati-hati link & lampiran email
4. Update software & antivirus
5. Backup data penting

🇮🇩 Di Indonesia

• Ada UU ITE yang ngatur cyber crime
• BSSN (Badan Siber Sandi Negara)
• Banyak komunitas hacker etis kayak IDSEC, IndoHack
• Butuh 10.000+ tenaga cyber security tiap tahun
